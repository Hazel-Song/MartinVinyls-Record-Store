# Martin Vinyl's Record Store — 设计风格

## 设计理念

整个项目围绕一个核心意象：**一家昏暗、略显颓败却充满温度的复古黑胶唱片店**。设计语言服务于叙事——用户不是在"使用一个 App"，而是在"走进一家店"。所有视觉、音效、交互细节都在强化这种沉浸感。

核心设计关键词：**复古 / 像素 / 昏暗 / 温暖橙光 / 颓废美学 / 仪式感**

## 色彩系统

### 主色调

| 角色 | 色值 | 用途 |
|------|------|------|
| 主强调色 | `#ff9800` / `rgb(255, 153, 0)` | 可交互元素、高亮标签、按钮、波形、选中状态 |
| 深强调色 | `rgb(200, 120, 0)` | 标签面板标题、边框线 |
| 悬停色 | `#f57c00` | 发送按钮悬停态 |

### 背景色阶

```
#000000  ← 页面底色（纯黑，防止背景图加载前闪白）
#2f3136  ← 聊天历史区背景（Discord 风暗色）
#36393f  ← 对话框背景（半透明叠加）
#40444b  ← 输入框背景、清除按钮背景
#202225  ← 聊天区边框、滚动条轨道
#333333  ← 标签词条背景
```

### 文字色

```
#dcddde  ← 主要文字（浅灰，高可读性）
#72767d  ← 占位符文字（低对比度，弱化引导）
#ffffff  ← 聊天消息文字、输出页标题
```

### 色彩逻辑

整个配色方案模拟了**昏暗店铺中一盏橙色台灯**的光感：
- 大面积深色（黑/深灰）= 昏暗的店铺空间
- 橙色高亮 = 台灯光晕，照亮重要信息
- 半透明叠加（`rgba`）= 玻璃/烟雾质感，增加层次

## 字体系统

### 唯一字体：IPix（自定义像素字体）

```css
@font-face {
    font-family: 'IPix';
    src: url('fonts/IPix.ttf') format('truetype');
}
```

**IPix 是一款像素点阵风格字体**，所有页面文字（包括 UI 标签、聊天内容、按钮文字）均使用此字体，无任何系统字体混用。

字体选择的设计意图：
- 像素字体天然带有"复古电子设备"的年代感，契合黑胶唱片的时代背景
- 统一字体消除了现代 UI 的"科技感"，强化了手工、老旧的氛围
- 中英文混排时，像素字体的等宽特性使排版更整齐

### 字号层级

| 层级 | 字号 | 使用场景 |
|------|------|----------|
| 超大标题 | `2.5em` | 输出页唱片名称 |
| 大标题 | `1.67em` | 输出页店铺名、创作信息 |
| 正文 | `1.2em` | 播放器曲目列表 |
| 小字 | `0.8em` | 输出页角落文字 |
| 基础 | 继承 | 聊天消息、输入框 |

## 视觉层次结构

页面采用严格的 `z-index` 分层，模拟真实空间的前后景关系：

```
z-index 层级（从后到前）：

z: 1   背景图层（.background）
z: 2   波形 Canvas（.wave-canvas）
z: 3   装饰图层（.sample-image 浮动封面）
z: 5   底部装饰图（.check-image）
z: 8   主容器（.container）
z: 10  输出页文字信息
z: 16  清除按钮
z: 50  推荐提示框（.recommend-tip）
z: 9999 导航按钮（.nav-buttons）
```

**设计意图**：推荐提示框（z:50）需要覆盖所有内容但不遮挡导航，导航（z:9999）始终在最顶层确保可操作性。

## 布局策略

### 全屏固定布局

所有主要元素使用 `position: fixed`，以视口为坐标系定位，不随内容滚动。这是一个**全屏沉浸式应用**，而非传统文档流页面。

```css
/* 典型固定布局模式 */
.element {
    position: fixed;
    top: X%;      /* 或 bottom: Xpx */
    left: X%;     /* 或 right: Xpx */
    width: Xvw;
    height: Xvh;
}
```

### 视口相对单位

大量使用 `vw`/`vh` 单位，确保在不同屏幕尺寸下比例一致：

```
NPC 角色：left: 20vw, bottom: 30vh, height: 60vh, width: 35vw
对话框：right: 390px, bottom: 70px, width: 40vw
波形 Canvas：width: 75vw, height: 400px
```

### 三页布局差异

| 页面 | 布局特点 |
|------|----------|
| `index.html` | NPC 居左，对话框居右下，标签面板顶部横排 |
| `player.html` | 左侧曲目列表（25vw黑色遮罩），右侧唱片机居中 |
| `output.html` | 封面图绝对居中，文字信息底部左对齐，角落信息左下角 |

## 动画系统

### 1. NPC 浮动动画

```css
@keyframes floatNPC {
    0%   { transform: translate(-50%, 50%); }
    50%  { transform: translate(-50%, calc(50% - 10px)); }
    100% { transform: translate(-50%, 50%); }
}
/* 周期：3s，缓动：ease-in-out，无限循环 */
```

**状态控制**：对话进行中动画播放，NPC 说完结束语后添加 `.paused` 类暂停动画，强化"店主停下来专心找唱片"的叙事感。

### 2. 唱片封面浮动动画

```css
@keyframes float {
    0%   { transform: translate(-50%, -50%); }
    50%  { transform: translate(-50%, calc(-50% - 10px)); }
    100% { transform: translate(-50%, -50%); }
}
/* 周期：3s，与 NPC 动画同步 */
```

输出页的唱片封面图使用相同节奏的浮动，营造"唱片漂浮在空中等待被取走"的仪式感。

### 3. Canvas 波形动画

三条叠加正弦波，参数各异，产生有机的层叠感：

```
波形 1：频率低(0.02)，振幅小(20px)，速度慢(0.05)，不透明度 0.9
波形 2：频率中(0.03)，振幅中(30px)，速度中(0.07)，不透明度 0.8
波形 3：频率高(0.04)，振幅大(40px)，速度快(0.09)，不透明度 0.7
```

**播放/暂停状态切换：**
- 播放中：振幅随机扰动（±2px/帧），范围 20~60px，动画速度 ×3 → 活跃、律动
- 暂停时：振幅衰减至接近 0，动画速度 ×0.1 → 平静、沉寂

### 4. 背景视差效果

```javascript
// 鼠标移动时背景轻微位移，系数 0.01（极小，仅产生微妙纵深感）
const moveX = (e.clientX - window.innerWidth / 2) * 0.01;
const moveY = (e.clientY - window.innerHeight / 2) * 0.01;
background.style.transform = `translate(calc(-50% + ${moveX}px), calc(-50% + ${moveY}px))`;
```

### 5. 过渡动画

```css
/* NPC 状态图切换（表情变化）*/
.npc { transition: background-image 0.3s ease; }

/* 背景视差移动 */
.background { transition: transform 0.5s ease-out; }

/* 曲目文字颜色变化 */
.track-text { transition: color 0.3s ease; }

/* 发送按钮颜色变化 */
#sendBtn { transition: background-color 0.2s; }
```

## 图像资源风格

### 背景图

三张背景图对应三个页面，均为像素风手绘场景：
- `background.png`：主页，唱片店内景，昏暗暖光
- `background-blur.png`：播放器页，同场景模糊处理，降低视觉干扰
- `background-items.png`：输出页，陈列台特写，物品感更强

### NPC 角色图

5 张状态图，像素风人物插画，通过 CSS `background-image` 切换，`transition: 0.3s` 平滑过渡。

### 唱片封面叠加图

7 张对应各音乐风格的唱片封面，作为 `<img>` 叠加在唱片机底座图上，选曲时动态切换 `src`。

### 导航按钮

图片按钮（非文字），整体缩放至 0.4 倍，以极小尺寸存在于右上角，不干扰主视觉，但始终可访问。

## 音效设计

音效是设计系统的重要组成部分，与视觉共同构建沉浸感：

| 场景 | 音效 | 音量 | 说明 |
|------|------|------|------|
| 对话页背景 | Echoes of Vinyl（环境音） | 0.3 | 低音量，不干扰对话 |
| 播放器试听 | 各风格样带 | 1.0（默认） | 主动试听，全音量 |
| 输出页背景 | 推荐歌曲 | 0.3 | 低音量，配合展示氛围 |

**自动播放策略**：所有页面均尝试自动播放背景音乐，被浏览器拦截时注册一次性 `click` 事件监听器，用户首次点击页面任意位置即解锁播放。

## 整体设计风格总结

Martin Vinyl's Record Store 的设计是一次**叙事驱动的沉浸式界面设计**实践：

1. **克制的 UI**：导航缩小至 0.4 倍，按钮用图片替代文字，最小化"软件感"
2. **统一的像素美学**：从字体到图像全部像素风，风格高度统一
3. **橙色作为情感锚点**：橙色不仅是强调色，更是"温暖、怀旧、人情味"的情感符号，贯穿所有可交互元素
4. **动画服务叙事**：NPC 浮动暂停、波形随音乐起伏，每个动画都有叙事意义而非纯装饰
5. **层次感营造空间**：多层半透明叠加 + z-index 分层，在二维屏幕上模拟三维店铺空间
