## 📼 Martin Vinyl's Record Store

![马丁黑胶的介绍](https://github.com/user-attachments/assets/f8c9229e-6284-4895-b17a-b2e7fe2cacac)

> 一家只为你开的复古黑胶唱片店。走进来，和店主聊聊天，让他为你挑一张唱片。

---

### 这是什么

一个 AI 驱动的沉浸式音乐推荐体验。

你不需要填问卷，不需要选标签，不需要告诉它你"喜欢什么类型"。你只需要和店主聊聊今天发生了什么、你现在的心情、那些说不清楚的感觉——然后，他会为你挑一张唱片。

---

### 认识马丁·黑胶

55 岁，郁郁不得志的老派唱片店主。见过太多带着各种心情走进来的人。他不给鸡汤，不假装热情，但他真的在听你说话。说话简短、犀利，偶尔讽刺，但总能说到点子上。

他的货架上全是私藏——那些你在任何平台都找不到的虚构曲目。

---

### 体验流程

```
走进店里  →  和马丁聊天  →  他为你挑唱片  →  带走它
```

**对话** — 聊什么都行。马丁的回复里会出现橙色高亮词，点击收集，那是他对你的理解。

**试听（可选）** — 不确定想要什么风格？去试听室转转，7 种风格的样带，一台老式唱片机，随音乐跳动的波形。

**拿走你的唱片** — 标签积累够了，马丁会说"好的，我在为你寻找合适的唱片，请稍等……"。走到礼品包装台，它在等你。

---

### 功能亮点

**情感理解，而非偏好匹配**
AI 从对话中提取三类标签——心情、风格偏好、故事背景——综合分析你当下的情绪状态，而不是你历史上喜欢什么。

**活的 NPC，不是聊天机器人**
马丁有完整的人设和说话方式，5 种表情状态随对话动态切换，会在合适的时机主动引导你走向推荐。

**仪式感贯穿始终**
从走进店里到拿走唱片，每一步都有设计：视差背景、NPC 浮动动画、Canvas 波形可视化、唱片封面的浮动展示。这是一次叙事体验，不是一个填表流程。

**像素复古美学**
自定义像素字体、手绘像素风图像、橙色作为唯一强调色——模拟昏暗店铺中一盏台灯的光晕。

---

### 技术栈

纯前端实现，零依赖：

- Vanilla JavaScript (ES6+)
- HTML5 / CSS3
- Canvas 2D API（波形动画）
- Web Audio API（音频播放）
- OpenAI GPT-4（NPC 对话 + 情感分析 + 风格推荐）
- localStorage（跨页面状态持久化）

---

### 本地运行

需要通过本地服务器运行（直接打开 HTML 文件会因跨域限制导致字体/音频加载失败）：

```bash
# 使用 VS Code Live Server 插件，或：
npx serve .

# 然后在浏览器打开 http://localhost:3000
```

在 `script.js` 和 `songRecommendation.js` 中替换你自己的 OpenAI API Key。

> ⚠️ 注意：当前 API Key 硬编码在客户端，仅供 Demo 使用。生产环境请迁移至后端代理。
